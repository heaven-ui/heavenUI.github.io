
## 配置选项
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|-|-|-|-|-|
| currentPage | 当前页 | Number | - | 1 |
| pageCount | 分页总页数 | Number | - | - |
| pagerNum | 页码显示数量，超过此数量会省略 | Number | 大于等于 5 且小于等于 21 的奇数 | 7 |
| prev | 上一页文本 | String | - | 上一页 |
| next | 下一页文本 | String | - | 下一页 |
