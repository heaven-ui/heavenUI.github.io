#### Input 输入框


#### 常用输入框。


### 配置选项
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|-|-|-|-|-|
| type | 类型 | String | text / textarea / 其他 原生 input 的 type 值 | text |
| value / v-model | 绑定值 | string / number | - | - |
| loading | 是否加载中状态 | Boolean | — | false |
| disabled | 是否禁用状态 | String | — | — |
| readonly | 设置输入框为只读 | Boolean | — | false |
| placeholder | 占位文本 | String | — | — |
