#### Icon 图标
提供了一套常用的图标集合。

``` html
<template>
  <h-icon name="loading"><h-icon>
</template>
```
``` js
<script>
export default {
  data() {
    return {
    }
  },
  methods: {
  },
};
</script>
```

## 配置选项
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|-|-|-|-|-|
| name | 图标的名称 | String | - | - |
| size | 图标的大小，单位是 px | String Number | - | - |
