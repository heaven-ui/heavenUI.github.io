#### 常用输入框。


### 配置选项
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|-|-|-|-|-|
| type | 类型 | String | default / primary / success / warning / danger / info | default |
| size | 尺寸 | String | mini / small / normal / large | normal |
| loading | 是否加载中状态 | Boolean | — | false |
| disabled | 是否禁用状态 | String | — | — |
| icon | 图标类名 | Boolean | — | false |
